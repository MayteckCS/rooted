# rooted
Spiritual Gifts Q&amp;A for Rooted


README: Dynamic Activation Diagnostic (Rooted Edition)
Project Overview
This tool is a "Scenario-Based Diagnostic" designed to complement the Mariners Church Rooted curriculum. Unlike traditional spiritual gift inventories that rely on historical self-reporting (which can be biased by experience or lack thereof), this tool measures real-time spiritual instinct.

The Problem with Static Testing
Traditional tests (like the Gifted2Serve model) often ask participants to rate themselves on a scale of 1-5 regarding past behaviors. For a new believer or someone in a "dry season," this can lead to:

Discouragement: Feeling "unspiritual" because they lack a history of ministry.

Performance Bias: Answering what they think a leader wants to hear.

Label Fatigue: Focus on the "title" of a gift rather than the "movement" of the Spirit.

The Solution: The "Scenario-Flow" Model
This prototype uses 20 real-world scenarios common to the Mariners Church environment (the lobby, small groups, local outreach).

1. The Four-Option Logic
Every question offers a specific "Activation Path":

Option A (Practical/Stabilizer): Measures the instinct for Practical Grace and Service.

Option B (Sensitive/Watchman): Measures the instinct for Intercession and Discernment.

Option C (Active/Catalyst): Measures the instinct for Manifestation Gifts (1 Cor. 12).

Option D (Stillness/Observer): A neutral baseline to ensure no one is forced into a "performance" answer.

2. Weighted Scoring Engine
The results are not a simple "one-to-one" count. The JavaScript engine uses weighted multipliers:

Catalyst Flow: Triggered by high "C" responses, indicating a readiness for power gifts (Faith, Healing, Miracles).

Watchman Flow: Triggered by "B" responses, indicating a strength in the "Revelation" gifts (Wisdom, Discernment).

Stabilizer Flow: Triggered by "A" responses, indicating a primary grace for "Helps" and "Administration."

Deployment Instructions
File Structure: The entire tool is contained within a single index.html file. No external databases or servers are required.

Usage: Open the file in any modern web browser.

Privacy: All data is processed locally in the user's browser. No personal data is stored or transmitted, aligning with a "Cautious" approach to digital privacy.

Strategic Alignment
This tool is designed to move members through the Mariners Loop:

Follow: Identifying the primary nudge of the Spirit.

Grow: Moving from a "B" (Observer) to a "C" (Active) response.

Serve: Mapping "Stabilizer" profiles to high-capacity service roles.

Change: Identifying "Catalysts" for frontier ministry and church planting.
